# Tasks App - User Guide

A personal productivity app for managing tasks across life domains.

---

## 1. Getting Started

### Overview
Tasks organizes your to-dos into 9 life domains: Body, Space, Mind, Plan, Connect, Attack, Create, Learn, and Manage. Each domain groups related tasks together for focused productivity.

### First Steps
1. Open the app - you'll see all 9 domains listed vertically
2. Each domain shows a header with its name and task count
3. Use the "+" button in the header to add your first task

---

## 2. Managing Tasks

### 2.1 Creating Tasks

#### From the Header (Global Add)
1. Click the "+" button in the top header bar
2. Fill in the task title (required)
3. Select which domain the task belongs to
4. Optionally set priority, effort, and complexity (1-3 scale)
5. Optionally pick due date and scheduled date using the calendar
6. Click "Add Task"

#### Inline Within a Domain
1. Click the "+" icon next to any domain header
2. Type your task title
3. Click "Add details" to expand metadata options
4. Set priority/effort/complexity and dates as needed
5. Click "Add Task"

### 2.2 Editing Tasks
1. Click on any task row to open the edit drawer
2. Modify title, domain, priority, effort, complexity, or dates
3. Click "Save" to apply changes
4. Click "Cancel" to discard changes

### 2.3 Completing Tasks
1. Click the checkbox circle on the left side of any task
2. The task moves to the "Completed" filter view
3. Completed tasks show a checkmark and completion timestamp

### 2.4 Reopening Tasks
1. Switch to the "Completed" filter
2. Click the checkbox circle on a completed task
3. The task returns to "Open" status

### 2.5 Archiving Tasks
1. Open the task edit drawer by clicking a task
2. Look for the archive option in task actions
3. Archived tasks are hidden from normal views

### 2.6 Restoring Archived Tasks
1. Switch to the "Archived" filter
2. Click on an archived task to open the edit drawer
3. Click the "Restore" button
4. The task returns to "Open" status

## 3. Organizing Tasks

### 3.1 Filtering Tasks
Use the filter bar at the top to show:
- **All** - Both open and completed tasks (default view)
- **Open** - Active tasks you're working on
- **Completed** - Tasks you've finished
- **Archived** - Tasks you've hidden away (preserves their open/completed status)

### 3.2 Sorting Tasks
Use the sort dropdown to order tasks by:
- **Manual** - Custom order you set via drag-and-drop
- **Due Date** - Tasks due soonest first
- **Scheduled** - Tasks scheduled soonest first
- **Priority** - Highest priority first (3 â†’ 1)
- **Effort** - Highest effort first
- **Complexity** - Highest complexity first
- **Created** - Newest tasks first

### 3.3 Drag-and-Drop Reordering
When in "Manual" sort mode with "Open" filter:
1. Grab the drag handle on the left side of a task
2. Drag the task up or down within its domain
3. Or drag to a different domain to move it there
4. Release to drop - the new order saves automatically
5. A "Saving..." indicator appears briefly during save

---

## 4. Task Metadata

### 4.1 Priority (1-3)
- **1** - Low priority, can wait
- **2** - Medium priority (default)
- **3** - High priority, do soon

### 4.2 Effort (1-3)
- **1** - Quick task, minimal effort
- **2** - Moderate effort (default)
- **3** - Significant effort required

### 4.3 Complexity (1-3)
- **1** - Simple, straightforward
- **2** - Some complexity (default)
- **3** - Complex, requires focus

### 4.4 Dates
- **Due Date** - When the task must be completed by
- **Scheduled Date** - When you plan to work on the task

---

## 5. Theme

### Switching Themes
1. Click the theme toggle button in the header
2. Switch between light and dark modes
3. Your preference is remembered

## 6. Domains

### Available Domains
The app comes with 9 pre-configured life domains:
1. **Body** - Health, fitness, physical well-being
2. **Space** - Home, environment, organization
3. **Mind** - Mental health, meditation, reflection
4. **Plan** - Planning, goals, strategy
5. **Connect** - Relationships, social, communication
6. **Attack** - Active work, projects, execution
7. **Create** - Creative work, building, making
8. **Learn** - Education, skills, knowledge
9. **Manage** - Administration, maintenance, logistics
